# AI Panel - 轻量级服务器管理面板

基于 Go 语言开发的现代化 Linux 服务器管理面板，单文件部署，无需运行时依赖。

> 最新发布版本：**v1.3.5**

## 功能特性

### 系统监控
- 实时 CPU、内存、磁盘使用率可视化
- 系统负载、网络流量统计
- 核心服务运行状态一览

### 网站管理
- OpenResty (Nginx) 站点管理，可视化配置
- **AI Nginx 配置助手**（智能分析配置、一键生成反代/缓存/SSL 优化，对话式修改）
- **网站日志分析**（全维度可视化仪表盘，4 个分析 Tab）
  - 流量概览：HTTP 方法分布、资源类型统计、24h 热力图、最耗带宽 URL
  - 错误排查：详细状态码表格、安全威胁检测
  - 性能分析：响应时间/大小分布、上游性能指标
  - 访客统计：爬虫 vs 人类、来源域名、User-Agent 分类
- **AI 日志智能分析**（SSE 流式输出专业分析报告）
- **CC 攻击检测引擎**（多因子风险评分：频率/路径集中度/Referer/UA）
  - 高危 IP 自动识别、被攻击路径关联、5 级风险等级
  - AI 直接给出封禁命令和 Nginx 防御配置
- SSL 证书自动申请（Let's Encrypt），支持 IP 证书
- PHP 多版本切换（7.4 / 8.0 / 8.1 / 8.2 / 8.3）

### 软件管理
- 一键安装 / 卸载 / 更新 / 启停
- **AI 软件安装助手**（分析系统环境，推荐版本，一键执行安装命令）

### Docker 管理
- 容器管理、镜像、Compose、网络和存储卷
- **AI Compose 生成**（自然语言生成 docker-compose.yml）

### 文件管理
- 在线文件浏览器、代码编辑器
- AI 代码编辑 / AI 网页生成 / AI 安全检测

### 数据库管理
- MySQL 管理 + **AI SQL 助手**
- Redis 键值浏览、内存监控

### AI 助手
- 集成 OpenClaw，支持多 Provider（Gemini/Claude/GPT/DeepSeek/Groq 等）
- 全模块 AI 助手（MySQL/Cron/防火墙/Docker/终端/系统优化/Nginx/日志分析）
- SSE 流式输出、聊天记录持久化、全局浮窗

### 安全功能
- AES-256-GCM 加密、2FA 认证、IP 白名单、防火墙管理
